## 介绍

rust-webook 是宇润用 Rust 开发的简易 Webhook 服务，支持 Gitee 和 Github。

可执行文件下载：<https://github.com/Yurunsoft/rust-webhook/releases>

PHP Swoole 版本：<https://gitee.com/yurunsoft/swoole-webhook>

> 两个项目的配置文件通用
